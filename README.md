# LoanEligibilityLINE
A full-stack web application that predicts loan eligibility using Machine Learning. Users can register, submit financial details, calculate EMI, and view eligibility scores with AI-based insights.  Tech Stack: React (Vite), Node.js, Express, MongoDB, Python Flask, Scikit-learn.  Architecture: Frontend (5173) → Backend API (5000) → ML Service (6000)
